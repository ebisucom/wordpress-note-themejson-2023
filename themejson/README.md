# theme.jsonの設定

theme.jsonの設定を記述したファイルです。
ファイル名を「theme.json」にして利用してください。

| ファイル                 | 内容 |
| ----------------------- | ------------------- |
| theme-all.json          | 設定をまとめたもの |
| theme-empty.json        | 空のtheme.jsonの設定 |
| theme-width.json        | 横幅を指定する設定 |
| theme-blockgap.json     | 間隔を有効化する設定 |
| theme-padding.json      | 左右の余白を指定する設定 |
| theme-rootpadding.json  | 左右の余白を全幅対応にする設定 |
| theme-ui-all.json       | エディターのUIをまとめて有効化する設定 |
| theme-ui-each.json      | エディターのUIを個別に有効化する設定 |
| theme-other.json        | 対策で指定する各種設定 |
